== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/st-elementor-addons` directory or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Open the Elementor editor and search for widgets like "ST Button", "ST Marquee", or "ST Flexbox Carousel".
4. Drag and drop the widgets onto your page and customize them as needed.

== Frequently Asked Questions ==

= Does this plugin require Elementor? =
Yes, this plugin works as an addon for Elementor and requires Elementor to be installed and activated.

= Can I use this plugin with any theme? =
Yes, as long as the theme supports Elementor, this plugin will work seamlessly.

= Is this plugin translation ready? =
Yes, the plugin is fully translation ready. You can translate it into your language using .pot files.

= How do I use Dynamic Tags in this plugin? =
Dynamic Tags can be used within the plugin’s widgets by selecting the "Dynamic Tags" option from the widget settings. This allows you to pull dynamic content such as post titles, author names, custom fields, and more directly into your widgets.

= What is the Flexbox Carousel? =
The **Flexbox Carousel** is a powerful Elementor container-based carousel. You can easily add any Elementor widget inside the carousel, and it will automatically display as a draggable, scrollable slider. It’s flexible, responsive, and fully customizable.
